# 勤怠システム(A)

本アプリケーションは従業員の勤怠を管理するWebアプリケーションです。
勤怠システム(B)で作成したアプリケーションをベースに、機能を追加しました。

## 仕様書からの変更点

- ユーザー名曖昧検索、ページネーション、勤怠一覧の週表示機能
→ 勤怠Bから引き続き実装。
- 削除・閉じる・キャンセルのボタン色
→ 他ボタンと同色の場合、視認性が悪く誤操作の恐れがあるため該当ボタンを赤に変更
- ユーザー一覧における上長ユーザー強調、優先表示機能の実装
→ 上長ユーザーが他ユーザーに埋もれた場合誤操作の可能性があるため、該当ユーザーの名前の横に上長バッジを表示、上長ユーザーを一般ユーザーより先に表示

# 主な機能

### 一般ユーザー

- ログイン / ログアウト
- アカウント作成
- 個人情報編集
- 勤怠表示（月 / 週単位切り替え・前週/次週ナビゲーション）
- 勤怠一括編集（月単位）
- 出勤・退勤時間の記録
- 在社時間の自動計算（15分単位）
- 勤怠表の曜日色分け（土曜：青 / 日曜：赤）
- 残業申請（上長への申請・翌日またぎ対応）
- 勤怠変更申請（出退社時間の変更を上長へ申請）
- 所属長承認申請（月単位の勤怠承認を上長へ申請）
- 勤怠CSV出力（表示月の出社・退社時間をCSVダウンロード）
- 勤怠修正ログ表示（承認済みの変更履歴を確認）

### 上長ユーザー

- 残業申請のお知らせ（申請中・否認の件数表示、承認 / 否認）
- 勤怠変更申請のお知らせ（承認時に勤怠実績へ自動反映）
- 所属長承認申請のお知らせ（月単位の承認 / 否認）
- 申請者の勤怠確認（読み取り専用で申請者の勤怠ページを閲覧）
- 同一日・同月の再申請時は最新の申請のみ保持

### 管理者

- ユーザー一覧表示（アコーディオン形式での編集・ページネーション対応）
- ユーザー情報編集（名前・メール・所属・社員番号・カードID・パスワード・勤務時間・指定勤務開始/終了時間）
- ユーザー削除（自身および最後の管理者の削除を防止）
- ユーザーCSVインポート（一括登録・更新）
- 出勤社員一覧（当日出勤中の社員を表示）
- 拠点情報管理（一覧・追加・編集・削除）

### アクセス制御

- 一般・上長ユーザーは管理者ページにアクセス不可
- 管理者は一般・上長の勤怠表示・編集にアクセス不可

## 技術スタック

| 種別              | 技術                          |
| ----------------- | ----------------------------- |
| 言語              | Ruby 3.0.6                    |
| フレームワーク    | Rails 7.1                     |
| DB（本番）        | PostgreSQL（pg）              |
| DB（開発/テスト） | Docker (mysql2)               |
| フロントエンド    | Bootstrap 3, jQuery           |
| 認証              | bcrypt（has_secure_password） |
| インフラ          | Docker / Docker Compose       |

## データベース構成

| 環境               | アダプター      | 理由                              |
| ------------------ | --------------- | --------------------------------- |
| development / test | mysql2          | Docker で MySQL コンテナを使用    |
| production         | pg (PostgreSQL) | Heroku 等の本番環境に合わせた設定 |

## 主なテーブル

| テーブル名                 | 用途                               |
| -------------------------- | ---------------------------------- |
| users                      | ユーザー情報（一般・上長・管理者） |
| attendances                | 日次の勤怠記録                     |
| overtime_requests          | 残業申請                           |
| attendance_change_requests | 勤怠変更申請                       |
| approval_requests          | 所属長承認申請                     |
| attendance_correction_logs | 勤怠修正ログ（承認済み履歴）       |
| work_bases                 | 拠点情報                           |

## ローカル環境セットアップ

```
# リポジトリをクローン

git clone https://github.com/YoshitatsuSaita/Attendance_A
cd Attendance_a

# コンテナ起動
docker compose up -d

# DB 作成・マイグレーション・seed
docker compose exec web bin/rails db:create db:migrate db:seed

```

## テスト実行

```bash
$ docker compose exec web bundle exec rspec
```
